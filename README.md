# Inact Alert Website

A modern web application that describe the process to review alerts, built with React, Vite, Tailwind CSS, and Radix UI components.

## Features

- Responsive UI with Radix UI and Tailwind CSS
- Form validation using React Hook Form and Zod
- Data visualization with Recharts
- Carousel support via Embla
- Routing with React Router
- Theming support with Next Themes

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (recommended package manager)
- Node.js (if not using Bun)

### Installation

```sh
bun install
```

### Development

```sh
bun run dev
```

### Build

```sh
bun run build
```

### Lint

```sh
bun run lint
```

### Preview

```sh
bun run preview
```

## Project Structure

- `src/` — Main source code
- `public/` — Static assets
- `package.json` — Project dependencies and scripts

Create a `.env` file for any sensitive configuration. **Do not commit secrets to the repository.**
